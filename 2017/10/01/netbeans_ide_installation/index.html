NetBeans IDE Installation on Wondows

簡介

NetBeans是一款由Oracle公司開發的集成開發環境(Integrated Development Environment, IDE)。 IDE能夠輔助開發者軟體的開發，IDE中內建相關語言的函式庫、除錯器、編譯器、直譯器與程式碼編輯器等子工具。 使用IDE的好處在於開發者無須自行安裝多項輔助開發的工具， 可以節省開發者配置開發環境的麻煩，IDE提供的圖形化介面也能提供開發者較直覺的功能使用。 本篇將引導讀者安裝用來撰寫Java的IDE－NetBeans，當然NetBeans也支援多項語言的開發，包含C/C++、Python與PHP等程式語言。

本篇的重點將告訴讀者如何安裝NetBeans IDE：

下載準備

  1. NetBeans IDE

安裝流程

Step1

前往Oracle官方網站，點選NetBeansw的DOWNLOAD前往下載安裝執行檔的網站。

Step2

確認並同意License後，選擇下載與系統環境相符的安裝執行檔，小編的環境Windows x64，所以選擇jdk-8u141-nb-8_2-windows-x64.exe。

Step3

找到電腦中下載完成的安裝執行檔，雙點擊該安裝執行檔。

Step4

點擊Next開始設定安裝配置。電腦中如果之前沒有安裝過Java Developmet Kit(JDK)， NetBeans安裝檔中事實上也包含JDK的安裝，而小編的電腦中已經安裝過JDK了，所以安裝程式顯示在小編的電腦上有偵測到JDK。

Step5

選擇並確認JDK的安裝路徑(預設路徑即可)後，點選Next開始執行安裝。

Step6

接著確認所有安裝配置的設定後，點擊Install開始安裝。

Step7

等待程式安裝完成。

Step8

安裝完成後點擊Finish，之後便可以開始使用NetBeans開發Java程式了。
