Java Developmet Kit(JDK) 8 Installation on Wondows

目錄

  • 簡介
  • 下載準備
  • 安裝流程

簡介

由Java撰寫的任何程式都需要運行在具有Java虛擬環境的電腦上，這也是Java的跨平台性質的來由， 所以在運行任何Java程式之前都必須安裝Java Runtime Environment(JRE)， 而Java Development Kit(JDK)則是Oracle公司提供給Java開發者配置Java程式開發環境的安裝套件， 這個安裝套件本身就內建了JRE的部分，所以開發者只要安裝好JDK後，就可以開發或者行任何Java程式了。

本篇的重點將告訴讀者如何安裝JDK。

下載準備

  1. Java Development Kit(JDK) 8

安裝流程

Step1

前往JDK 8官方網站，選擇下載與系統環境相符的安裝執行檔，小編的環境是Windows x64，所以選擇jdk-8u144-windows-x64.exe。

Step2

找到電腦中下載完成的安裝執行檔，雙點擊該安裝執行檔。

Step3

點擊Next開始設定安裝配置。

Step4

選擇並確認JDK的安裝路徑後，預設路徑即可，點選Next開始執行安裝。

Step5

等待安裝程式安裝完成。

Step6

接著程式會進行JRE的安裝，選擇並確認JRE的安裝路徑，預設路徑即可，點選下一步開始執行安裝。

Step7

等待安裝程式安裝完成。

Step8

完成安裝後，按下Close完成安裝。
